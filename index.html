Utilización de Sass

Preprocesador más antiguo de Css que permite crear hojas de estilos con una mejor estructura y reducción de líneas de código mediante vars, each, function, etc.

Sass parte II

Sass ofrece varios métodos para trabajar con los elementos. Condicionales, bucles, funciones, map (listas), @extend, @mixin (similar a una función), etc. Consecuentemente a favor reducirá las líneas de código y mejora la estructura de la hoja de estilos.
